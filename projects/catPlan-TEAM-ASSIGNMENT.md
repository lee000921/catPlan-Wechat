#### 技术细节

**SSL 证书申请**:
- 阿里云 SSL 服务（免费 DV 证书）
- 域名：`catplan.xin`
- 有效期：1 年

**Nginx 配置**:
```nginx
server {
    listen 443 ssl;
    server_name catplan.xin;
    
    ssl_certificate /etc/nginx/ssl/catplan.xin.crt;
    ssl_certificate_key /etc/nginx/ssl/catplan.xin.key;
    
    location /api {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}

server {
    listen 80;
    server_name catplan.xin;
    return 301 https://$server_name$request_uri;
}
```
