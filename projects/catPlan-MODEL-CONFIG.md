## 🔧 配置方式

### OpenClaw 配置

```json5
{
  models: {
    providers: {
      dashscope: {
        // 阿里云百炼配置
        baseUrl: "https://dashscope.aliyuncs.com/compatible-mode/v1",
        apiKey: "__OPENCLAW_REDACTED__",
        models: [
          {
            id: "qwen3.5-max",
            name: "Qwen3.5-Max",
            cost: { input: 0.014, output: 0.042 }
          }
        ]
      },
      zhipu: {
        // 智谱 AI 配置
        baseUrl: "https://open.bigmodel.cn/api/paas/v4",
        apiKey: "__OPENCLAW_REDACTED__",
        models: [
          {
            id: "glm-5",
            name: "GLM-5.0",
            cost: { input: 0.005, output: 0.020 }
          }
        ]
      }
    },
    agents: {
      defaults: {
        // 默认研发模型
        model: "zhipu/glm-5"
      },
      list: [
        {
          id: "main",
          model: "dashscope/qwen3.5-max"  // TL 使用最强模型
        },
        {
          id: "reviewer",
          model: "dashscope/qwen3.5-max"  // 审查使用最强模型
        },
        {
          id: "developer",
          model: "zhipu/glm-5"  // 开发使用 GLM-5.0
        }
      ]
    }
  }
}
```

### 子代理配置

```bash
# 后端开发（扳手、鲁班）
/subagents spawn developer "实现 XXX 功能" --model zhipu/glm-5

# 前端开发（画眉、丹青）
/subagents spawn developer "开发 XXX 页面" --model zhipu/glm-5

# 代码审查（天眼）
/subagents spawn reviewer "审查 XXX 代码" --model dashscope/qwen3.5-max

# 自动化部署（闪电）
/subagents spawn developer "编写部署脚本" --model zhipu/glm-5
```

## ⚠️ 注意事项

### 模型切换

1. **开发过程中**如发现 GLM-5.0 处理不了的问题，可临时切换到 Qwen3.5-Max
2. **代码审查**必须使用 Qwen3.5-Max，确保审查质量
3. **对外交付**的代码/文档，建议使用 Qwen3.5-Max 最终审核

### 成本控制

1. 每日查看 token 使用情况
2. 复杂任务优先使用 GLM-5.0
3. 关键决策使用 Qwen3.5-Max
4. 定期审查模型使用效率

### 质量保障

1. GLM-5.0 生成的代码需经过天眼（Qwen3.5-Max）审查
2. 关键功能需经过小白（GLM-5.0）测试
3. 重要决策需狗头老大最终确认
