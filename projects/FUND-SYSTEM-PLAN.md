**版本**: v1.0  
**创建日期**: 2026-03-05

## 📋 项目概览

### 项目目标

**构建一个仅供内部使用的基金估值和净值查询系统**

- 实时获取基金估值（盘中）
- 实时获取基金净值（盘后）
- 简单的数据展示界面
- 仅供少数几人内部使用

### 核心功能

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **基金估值** | 盘中实时估算净值 | 🔴 P0 |
| **基金净值** | 盘后实际净值 | 🔴 P0 |
| **基金列表** | 自选基金展示 | 🔴 P0 |
| **历史数据** | 历史净值查询 | 🟡 P1 |
| **涨跌提醒** | 涨跌幅提醒 | 🟡 P1 |
| **数据导出** | 导出 Excel | 🟢 P2 |

## 🏗️ 技术架构

### 整体架构

```
┌─────────────────┐
│   前端展示层    │  Streamlit (快速开发)
├─────────────────┤
│   后端 API 层    │  FastAPI (轻量级)
├─────────────────┤
│   数据接入层    │  Tushare + AkShare
├─────────────────┤
│   数据存储层    │  SQLite (轻量级)
└─────────────────┘
```

### 前端技术选型

| 技术 | 选型 | 说明 |
|------|------|------|
| **框架** | Streamlit | Python 快速开发，无需前端知识 |
| **部署** | 本地运行 | 无需服务器，本地访问 |
| **访问** | 浏览器 | http://localhost:8501 |

**为什么选 Streamlit？**
- ✅ 开发速度快（1 周完成前端）
- ✅ 无需前端知识（纯 Python）
- ✅ 适合内部工具（不需要精美 UI）
- ✅ 免费开源

### 后端技术选型

| 技术 | 选型 | 说明 |
|------|------|------|
| **框架** | FastAPI | 轻量级、高性能 |
| **语言** | Python 3.9+ | 数据生态丰富 |
| **数据库** | SQLite | 轻量级、无需配置 |
| **部署** | 本地运行 | 无需服务器 |

### 数据源选型

| 数据源 | 用途 | 成本 | 推荐度 |
|--------|------|------|--------|
| **Tushare** | 主力数据源 | ¥500/年 | ⭐⭐⭐⭐⭐ |
