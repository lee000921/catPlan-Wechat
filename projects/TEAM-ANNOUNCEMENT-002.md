### 使用方式

**子代理调用**（自动使用 GLM-5.0）：
```bash
# 后端开发
/subagents spawn developer "实现用户登录功能"
# 自动使用 dashscope-coding/glm-5

# 前端开发
/subagents spawn developer "开发任务列表页面"
# 自动使用 dashscope-coding/glm-5

# 自动化脚本
/subagents spawn developer "编写部署脚本"
# 自动使用 dashscope-coding/glm-5
```

**如需临时使用 Qwen3.5-Max**：
```bash
/subagents spawn reviewer "审查这段代码" --model dashscope-us/qwen3-max-2025-09-23
```
