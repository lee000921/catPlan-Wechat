## 🎯 核心改动说明

### 架构变化

#### 原架构（腾讯云）
```
小程序 → wx.cloud.callFunction() → 云函数 → 云数据库
```

#### 新架构（阿里云）
```
小程序 → wx.request() → Express API → MongoDB → 响应
                ↓
              Nginx
                ↓
          阿里云 ECS
```

### 主要优势
1. ✅ **成本更低** - 月费用约210-360元（vs 腾讯云CloudBase）
2. ✅ **更灵活** - 完全控制后端逻辑和数据库
3. ✅ **易调试** - 可以本地调试，查看详细日志
4. ✅ **可扩展** - 方便添加新功能和中间件
5. ✅ **通用性强** - Express技术栈，团队容易上手

## 📝 您需要做的事情

### 第一步：准备阿里云资源
1. 注册阿里云账号并实名认证
2. 购买/开通以下服务：
   - ECS云服务器（推荐：2核4G，3M带宽）
   - MongoDB数据库（或在ECS上自建）
   - 对象存储OSS（存储图片）
   - 域名（需备案）
   - SSL证书（Let's Encrypt免费）

### 第二步：部署后端
```bash
# 1. 连接到ECS服务器
ssh root@your-server-ip

# 2. 上传代码到服务器
cd /www
git clone your-repo.git catplan-backend
cd catplan-backend/backend-aliyun

# 3. 运行部署脚本
chmod +x deploy.sh
./deploy.sh

# 4. 配置.env文件
vim .env
# 填入：MongoDB地址、JWT密钥、微信AppID/Secret等

# 5. 导入数据
node scripts/importData.js

# 6. 启动服务
pm2 start ecosystem.config.js
pm2 save
pm2 startup

# 7. 配置Nginx和SSL
# 参考 ALIYUN_MIGRATION.md 中的Nginx配置
```

## 💰 成本对比

### 腾讯云 CloudBase
- 按量计费，容易超支
- 云函数调用费
- 数据库读写费
- 存储费用
- **预估：500-1000元/月**（活跃用户多时）

### 阿里云方案
| 服务 | 配置 | 月费用 |
|------|------|--------|
| ECS | 2核4G 3M | ¥100-150 |
| MongoDB | 自建/1核2G | ¥0-100 |
| OSS | 10GB | ¥10-30 |
| 域名 | .com | ¥50/年 |
| SSL | Let's Encrypt | 免费 |
| **合计** | - | **¥210-360/月** |

**节省：40-70%成本**

## 🔥 快速决策指南

### 继续使用腾讯云，如果：
- ✅ 已经熟悉腾讯云生态
- ✅ 用户量小（日活<1000）
- ✅ 不想管理服务器
- ✅ 快速上线（无需部署）

### 迁移到阿里云，如果：
- ✅ 想降低成本
- ✅ 需要完全控制后端
- ✅ 用户量大或快速增长
- ✅ 团队有Node.js经验
- ✅ 需要复杂的业务逻辑

## ⚡ 快速命令参考

### 本地开发
```bash
cd backend-aliyun
npm install
cp .env.example .env
# 编辑 .env 配置

```

### 服务器部署
```bash
# 运行部署脚本
./deploy.sh  # Linux
deploy.bat   # Windows

# PM2管理
pm2 start ecosystem.config.js
pm2 status
pm2 logs catplan-backend
pm2 restart catplan-backend
```

### 数据管理
```bash
# 导入数据
node scripts/importData.js

# 备份数据库
mongodump --uri="mongodb://..." --out=./backup

# 恢复数据库
mongorestore --uri="mongodb://..." ./backup
```

### Nginx
```bash
# 测试配置
nginx -t

# 重启
systemctl restart nginx

# 查看日志
tail -f /var/log/nginx/error.log
```
