# 前端代码规范

**适用范围**：catPlan-Wechat 前端项目

---

## 📋 目录结构

```
catPlan-Wechat/
├── miniprogram/
│   ├── pages/         # 页面
│   ├── services/      # API 服务
│   ├── utils/         # 工具函数
│   └── images/        # 图片资源
```

---

## 🎨 代码风格

### 命名规范

**文件命名**：小写字母，中划线连接，如 `task-list.js`

**变量命名**：小驼峰，如 `userInfo`

**函数命名**：小驼峰，动词开头，如 `getUserInfo()`

---

## 📝 提交规范

### Commit Message 格式

```
<type>(scope): <subject>

<body>

Co-authored-by: <花名> <角色>
```

### Type 说明

- feat: 新功能
- fix: Bug 修复
- docs: 文档更新
- style: 代码格式
- refactor: 重构
- test: 测试
- chore: 构建/工具

### 署名格式

```
Co-authored-by: 画眉 🎨
Co-authored-by: 丹青 ✏️
```
