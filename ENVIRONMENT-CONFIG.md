## 📝 提交流程

### Git 提交规范

```bash
# 格式
<type>(scope): <subject>

<body>

Co-authored-by: <花名> <角色>
```

### Type 说明

| Type | 说明 | 示例 |
|------|------|------|
| feat | 新功能 | feat(shop): 添加兑换功能 |
| fix | Bug 修复 | fix(auth): 修复登录验证 |
| docs | 文档更新 | docs: 更新 API 文档 |
| style | 代码格式 | style: 格式化代码 |
| refactor | 重构 | refactor: 重构用户模块 |
| test | 测试 | test: 添加单元测试 |
